### 项目技术栈

- Java 1.8
- Mybatis Plus 3.0.5
- SpringBoot 2.7.0
- Mysql 5.7
- Redis 5.7.0
- Swagger 2.9.0
- Jwt Token

------

### 项目概述

项目采用前后端分离的形式开发，这里只是一个后端项目的基础模块， 试用于单体项目的快速开发，基本的权限系统接口都有，用户只需要关注具体的业务模块即可。 目前项目会持续维护中。。。

### 项目功能

- 首页功能
- 用户管理
- 角色管理
- 菜单管理
- 岗位管理
- 部门管理
- 权限管理
- 日志管理
